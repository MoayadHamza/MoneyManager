# MoneyManager

A personal-finance tracker for recording income and expenses and seeing where the
money goes.

## Features

- Add, edit, and delete income/expense entries
- Categorize transactions and view totals per category 
- Summary of balance over time 


## Getting started

```bash
git clone https://github.com/MoayadHamza/MoneyManager.git
cd MoneyManager
pip install -r requirements.txt
python main.py
```
